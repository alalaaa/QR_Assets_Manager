# QR_Assets_Manager
The QR Assets Manager project aims to develop an advanced mobile application that automates the IT equipment inventory process within companies, replacing traditional and error-prone paper inventory sheets with a modern digital solution. The system is based on using mobile devices as terminals that, after scanning a dedicated QR code sticker on a piece of equipment, instantly retrieve and display the device's full technical specification, its current owner, and its service history from the database. To ensure high performance and reliability, the development team—consisting of Adam Ratajczyk, Alicja Zielińska, Krystian Chojnacki, Błażej Orłowski, Kacper Ciągło, and Rafał Borman—is utilizing a modern tech stack including a Laravel backend and a React TypeScript frontend. Key functionalities include rapid QR scanning with automatic LED flash support for low-light conditions, an advanced search engine with filters for brand, model, or location, and a mode for adding new assets directly from a smartphone.
