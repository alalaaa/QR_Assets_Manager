# QR_Assets_Manager
Projekt QR Assets Manager ma na celu stworzenie zaawansowanej aplikacji mobilnej, która zautomatyzuje proces ewidencji sprzętu komputerowego w przedsiębiorstwach, zastępując tradycyjne i podatne na błędy papierowe arkusze inwentaryzacyjne nowoczesnym rozwiązaniem cyfrowym. System opiera się na wykorzystaniu urządzeń mobilnych jako terminali, które po zeskanowaniu dedykowanej naklejki z kodem QR natychmiastowo pobierają z bazy danych i wyświetlają pełną specyfikację techniczną urządzenia, dane jego aktualnego właściciela oraz szczegółową historię serwisową.
